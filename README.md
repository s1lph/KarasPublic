# Karas Messenger - Безопасный шифрованный мессенджер 

Веб‑мессенджер (React + Vite) с поддержкой E2EE
Прод: https://directt.up.railway.app/

## Возможности

- Личные чаты и групповые чаты.
- Отправка сообщений, редактирование и удаление.
- Рабочие в РФ пересылка файлов и медиа.
- Звонки через WebRTC + signaling через Socket.IO; используется шифрованный иностранный сервер.
- Уведомления в браузере и Web Push (Работает на Firefox и Chrome).
- E2EE (сквозное шифрование) для личных чатов.
- Установка на Android/Ios через браузерную закладку.

## Как работает E2EE (ключи и шифрование)

Сквозное шифрование (E2EE) — это метод защиты данных, при котором сообщения, звонки и файлы шифруются непосредственно на устройстве отправителя и расшифровываются только на устройстве получателя. Посредники (серверы, провайдеры, хакеры) не имеют ключей для дешифровки, поэтому не могут перехватить или прочитать информацию, так как она передается в нечитаемом виде.

Важно: E2EE реализовано для личных диалогов (1‑на‑1). Для групповых чатов сквозное шифрование в текущей версии не используется.

## Что такое “бэкап ключа” и как он работает

Чтобы можно было зайти с нового устройства и не потерять доступ к E2EE‑сообщениям, есть опциональный бэкап приватного ключа на сервере:

- Чтобы подключить бэкап ключей — смените пароль в настройках(можно указать тот же самый).
- В мессенджере всегда будет отображаться аварийный баннер, если бэкап ключ не подключен

Если пароль введён неверно или бэкап повреждён, восстановление не произойдёт и E2EE на этом устройстве будет недоступно, пока ключ не появится локально. Все это из-за особенностей E2EE шифрования, для полной безопастности сообщений(в том числе от владельца сайта)

## Установка на смартфон в качестве веб-приложения(PWA)

Мессенджер устанавливается из браузера и открывается как отдельное приложение.

### Android, нативно для браузера Firefox

1. Откройте сайт в Firefox: https://directt.up.railway.app/
2. Нажмите меню (три точки).
3. Выберите “Install app” / “Установить приложение” или “Добавить на главный экран”.
4. Подтвердите установку.
5. Зайдите в настройки -> уведомления -> запросить и дайте разрешение.
6. Нажмите подробнее по приложению и в разделе батарея поставьте unrestricted (неограниченно).
7. Также рекомендуется отключить функцию "Остановить работу приложения при простое"

### iPhone/iPad, нативно для браузера Safari

1. Откройте сайт в Safari: https://directt.up.railway.app/
2. Нажмите кнопку “Поделиться” (Share).
3. Выберите “Add to Home Screen” / “На экран Домой”.
4. Подтвердите добавление.

Примечание: push‑уведомления в iOS работают только для установленного PWA и зависят от версии iOS и настроек системы.
